CSS/React Track

React is a JavaScript library used to create dynamic, interactive user interfaces. It allows a site to quickly update many different elements at once, without reloading the page. The CSS and React combination is perfect for people who want to build something fun, stylish, and create dynamic interactive user interafaces. It is on the front-end side of things.

Ruby/Rails Track

Ruby and Rails are typically used by younger companies and startups, especially for building interactive web applications. This is a great combo for those who would like to build interactive web applications (creating accounts, entering information, and interacting with dynamic content). These programs simplicity encourages productivity and has elegant syntax that is natural to read and easy to write. Generally more on the back-end side of things.

C#/.NET Track

C# and .NET are used by larger enterprises, government agencies, and companies serving enterprise and government clients for employment. These are great programs to know for those who would like to create client applications, database applications, distributed components, and much more. This combo's focus is on the back-end. It is complex but very robust and powerful.
